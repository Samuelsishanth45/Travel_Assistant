# 🚆 Journey Guider

Journey Guider is a Smart AI-based Travel Assistant built using:

* C++
* React.js
* Node.js
* Express.js
* Google Maps API
* Groq AI API

The project helps users explore and plan journeys across India using trains, buses, maps, tourist attractions, and AI smart planning.

---

# ✨ Features

## 1️⃣ Direct Train Finder

* Find direct trains between stations
* Covers 11,000+ Indian trains
* Shows timings, distance and duration

## 2️⃣ Bus Route Explorer

* Finds multiple bus routes
* Includes route suggestions and tips

## 3️⃣ Nearest Railway Stations

* Shows nearest stations from any location
* Includes Google Maps links

## 4️⃣ Smart AI Travel Planner

* Generates day-wise travel plans
* Includes attractions and maps routes
* Optimized using travel timings

## 5️⃣ Tourist Attractions Explorer

* Displays famous places in cities
* Includes descriptions and best visiting times

## 6️⃣ Google Maps Direction Integration

* Route directions between locations
* Direct maps integration

## 7️⃣ AI Journey Planner

* Direct route planning
* AI smart indirect route planning
* Multi-modal travel planning

## 8️⃣ Indirect Train Route Finder

* Finds routes with transfers
* Intelligent station switching
